# lanzar-hilos-desde-java

Proyecto de práctica de **concurrencia y multihilo en Java**.  
Contiene 10 ejercicios basados en pseudocódigo de hilos, convertidos a clases Java ejecutables con sus respectivos **tests JUnit 5**.

---

## ⚙️ Requisitos

- Java 17
- Maven 3.x
- IDE compatible con Maven (IntelliJ IDEA, Eclipse, VS Code...)
